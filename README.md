Objective
Build a minimal Task Manager application using the MERN stack (MongoDB, Express, React, Node.js).
Backend Requirements (Node.js + Express + MongoDB)
Implement the following APIs:
GET /tasks – Retrieve all tasks
POST /tasks – Create a new task
Task Schema
title (String, required)
completed (Boolean, default: false)
Guidelines
Use async/await
Add basic validation (title is required)
Return proper HTTP status codes
Organize code clearly
Frontend Requirements (React)
Create a simple interface that:
Displays tasks
Allows adding a new task
Guidelines
Use functional components with Hooks
Use Fetch or Axios for API calls
Keep the UI minimal and clean
Submission Instructions
Include a README file with setup instructions
